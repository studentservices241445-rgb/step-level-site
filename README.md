# STEP Level‑Test Site

هذا الموقع المجاني يساعد الطلاب على تقييم مستواهم في اختبار كفايات اللغة الإنجليزية (STEP) وتنظيم خطط المذاكرة الخاصة بهم. يتضمن:

* **اختبار محاكي** مكوّن من 50 سؤالاً يتم اختيارها عشوائياً من بنك أسئلة واسع.
* **تصحيح فوري** وإظهار التفسير لكل سؤال.
* **تحليل النتائج** حسب الأقسام (قراءة، استماع، قواعد، مفردات) وتوليد خطة مذاكرة شخصية تعتمد على نقاط الضعف وموعد الاختبار.
* **توصية بدورات** متنوعة لتقوية المهارات مع توفير نموذج تسجيل يشمل طرق دفع متعددة.
* **مساعد افتراضي** يجيب على استفسارات المستخدم.
* **إشعارات اجتماعية** تظهر نشاطات المستخدمين السابقين لتشجيع الآخرين.

للتشغيل محلياً، افتح `index.html` في متصفح حديث. جميع الملفات ثابتة ولا تتطلب خادم خلفي.
